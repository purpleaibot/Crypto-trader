# Crypto-Trader

An autonomous, modular cryptocurrency trading system designed for conservative growth and controlled risk.

## Features
*   **Monitoring Bot:** Multi-timeframe trend analysis with volatility filters.
*   **Analyze Agent (NanoClaw):** LLM-powered trade validation with "human-in-the-loop" learning.
*   **Trading Bot:** Precision execution with Limit Orders and Trailing Stops.
*   **Modern UI:** Responsive dashboard for mobile and desktop.
